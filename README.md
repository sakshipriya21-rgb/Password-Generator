# Password-Generator
This tool automatically generates a random and strong password.

## Features

* Custom password length
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

## Technologies Used

* Python

## How to Run

Clone the repository:

git clone <repository-url>

Navigate to the folder:

cd Password-Generator

Run:

python main.py

## Example Output

Enter password length: 12

Generated Password:
A@7kL9!qP2#x
